Set-up development environment - Visual studio code

  • Step 1: node.js, visual studio code

  • Step 3: Create a New Expo Project npx expo init MyNewExpoProjectApp
  • Step 4: Navigate to the Project Directory cd MyNewExpoProjectApp
  • Step 5: Start the Development Server: npx expo start
